 ‚Ä¢ `bash installer.sh`

 ‚Ä¢ `nano .env`

  - ketik ctrl + S
  - ctrl + X

 ‚Ä¢ `screen -S naya`

 ‚Ä¢ `bash start`

<details>
<summary><b>üîó Deploy Via Docker</b></summary>
<br>

‚Ä¢ `curl -sSL https://get.docker.com | sh`

 ‚Ä¢ `git clone https://github.com/naya1503/Naya-Userbot`

 ‚Ä¢ `cd Naya-Userbot`

 ‚Ä¢ `cp sample.env .env`

 ‚Ä¢ `nano .env`
  - isi vars .env API_ID, API_HASH, SESSION dan MONGO_URI
  - Jika sudah 
  - ketik ctrl + S
  - ctrl + X

 ‚Ä¢ `docker build . -t naya`

 ‚Ä¢ `docker run --name namalu --cpus 1.2 --memory 500m --env-file .env naya`
